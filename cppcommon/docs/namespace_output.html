Detailed Description

Easily change the text and text background color for console output.

Class will do two basic checks to determine if output file (stdout / stderr) supports modification.
For example, IDEs like Xcode do not support output modification and it does not ignore the control characters. This code will detect Xcode and ignore modification characters.

Note
Not all terminals support all modifiers, most modern terminals should ignore modifiers they don't support but I haven't tested nearly all the available terminals
